# ResumeIQ – AI Resume Analyzer

ResumeIQ is an AI-powered web application that analyzes resumes and provides intelligent suggestions to improve them. The goal of the project is to help students and job seekers optimize their resumes to increase their chances of getting shortlisted by recruiters and Applicant Tracking Systems (ATS).

The system allows users to upload their resume and receive feedback about skills, structure, and possible improvements.

---

## Problem Statement

Many job seekers struggle to create effective resumes that pass ATS systems and attract recruiters. Often resumes lack important skills, proper formatting, or relevant keywords required for job roles.

ResumeIQ solves this problem by automatically analyzing resumes and providing suggestions to improve them.

---

## Solution

ResumeIQ processes uploaded resumes and evaluates the content to identify missing skills, weak sections, and improvement opportunities. It then provides suggestions to help users enhance their resumes and make them more competitive for job applications.

## Installation and Setup

Clone the repository:

```
git clone https://github.com/rudrabhuva/ResumeIQ.git
```

Navigate to the project folder:

```
cd ResumeIQ
```

Install required dependencies:

```
pip install -r requirements.txt
```

Run the application:

```
python app.py
```

Open the browser and go to:

```
http://127.0.0.1:5000
```
